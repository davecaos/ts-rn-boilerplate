# React Native Typescript Boilerplate

## 📕 Stack

- React Native
- Typescript
- React Navigation
- Redux & Redux JS Toolkit
- Redux Sagas
- Axios
- Jest & RNTL
- Storybook
- Reactotron
- Sentry
- Error Boundary
- Custom Fonts
- Theme Context
- Eslint & Prettier
- Husky & Lint Staged

## 👍 Feel free to use this boilerplate

Fork or clone this repository and start working on your React Native project rightaway.
